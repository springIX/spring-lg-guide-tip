화질 차이를 만드는
TV 설정은 무엇일까요?

TV 화질이 아쉽다면, 설정부터 살펴보세요.
화질을 더 선명하게 만드는 간단한 설정 세 가지를 소개해요.

3줄로 요약해 드려요.

  • · 화면이 퍼지거나 잘려 보인다면, [전체 설정] → [화면] → [화면 크기 설정] → [원본 비율]을 선택해 주세요.

  • · 화면이 어둡게 보인다면, 영상모드를 [표준 영상]으로 설정하고 에너지 절약의 [절전모드]를 해제해 주세요.

  • · 인공지능이 적용된 LG TV라면 [인공지능 서비스] → [인공지능 화질 프로]를 활성화해 주세요.

화면 설정

화질을 개선하는
설정 3가지

간단한 설정으로 화질을
개선하는 방법을 알려드릴게요.

SETTING 01

화면 왜곡을 잡는 화면 크기 최적화

화면이 퍼지거나 잘린다면 확인해 보세요.

  • [전체 설정] → [화면] → [화면 크기 설정] →
    [원본 비율] 선택

* 소비자의 이해를 돕기 위해 연출된 이미지로, 실제 구현되는 화면과 다를 수 있습니다.

SETTING 02

영상모드와 절전 모드 점검

화면이 어둡고 답답하다면 에너지 절약
설정을 점검해 보세요.

  • 영상 모드

    [전체 설정] → [영상] → [영상 모드 설정]에서
    [표준 영상]으로 변경하세요.

  • 절전 모드

    [전체 설정] → [일반] → [에너지 절약]에서
    [절전 모드]를 [꺼짐]으로 설정하세요.

* 디지털 방송, HDMI 등 각 영상 입력 시 영상 모드 설정을 달리할 수 있으므로 입력 모드 변경이 있을 때마다 원하는 영상 모드(밝기 - 화질) 설정이 필요합니다.

* 소비자의 이해를 돕기 위해 연출된 이미지로, 실제 구현되는 화면과 다를 수 있습니다.

SETTING 03

AI 최적화 적용

인공지능이 적용된 LG TV라면 AI에게 화질 교정을
맡겨보세요.

  • AI 화질 프로

    [일반] → [인공지능 서비스] →
    [인공지능 화질 프로]를 [ON]으로 설정하세요.

  • 맞춤 화면 설정

    내 눈에 가장 편안한 화질을 찾아주는 기능입니다.
    (webOS23 이상 지원)

* 소비자의 이해를 돕기 위해 연출된 이미지로, 실제 구현되는 화면과 다를 수 있습니다.

* 인공지능이 적용된 TV는 2023년형 webOS23부터 지원됩니다.

* 연도별 webOS 버전에 따라 메뉴 경로가 다를 수 있으므로, 구입 시기를 확인하여 해당 방법을 참고하시기 바랍니다.

* LG전자의 TV는 매년 더 발전된 핵심 소프트웨어인 webOS가 제품에 설치됩니다.

외부 환경 설정

재생해 보세요. 이 영상이 선명하다면
TV 기기 문제가 아닌 방송 신호 문제일 수 있습니다.

셋톱박스 해상도

셋톱박스 설정 메뉴에서 출력 해상도가
2160p(UHD) 또는 최대로 되어 있는지 확인하세요.

신호 세기 체크

공청/유선 TV 시청 시
[채널 검색] → [수동 채널] 메뉴에서
신호 품질이 30% 미만이라면 관리사무소나
통신사의 점검이 필요합니다.

* 소비자의 이해를 돕기 위해 연출된 이미지로, 실제 구현되는 화면과 다를 수 있습니다.

TIP

  • 모든 설정을 마쳤는데도 화질이 이상하다면, LG TV의 화면 자가 진단 기능을 활용해 보세요.

  • [고객 지원] → [디바이스 셀프 케어] → 화면 자가 진단 서비스를 받아보세요.
